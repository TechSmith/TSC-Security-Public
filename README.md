# TechSmith Security Publications

This repository contains public security information published by the TechSmith Security Team.

The repository serves as a version-controlled location for security advisories, vulnerability-related statements, technical security publications, and other security communications that may be useful to customers, researchers, partners, and the broader security community.

## Additional Security Information

Compliance documentation, security policies, certifications, and related information are available through the TechSmith Trust Center:

https://trust.techsmith.com/

## Published Advisories

See the Advisories Index for published advisories and notices.

## Reporting Security Issues

Software vulnerabilities should be reported through the TechSmith Vulnerability Disclosure Program:
https://www.techsmith.com/trust-center/report-security-vulnerabilities/

For other security concerns, please submit a support request using the Security ticket type:
https://support.techsmith.com/

**Do not use GitHub Issues here to report security vulnerabilities.**

## Disclaimer

Information contained in this repository is provided on an as-is basis and may be updated as new information becomes available.

Publication of information in this repository does not imply product vulnerability, product impact, or acceptance of third-party claims unless explicitly stated.
