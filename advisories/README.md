# Security Advisories

This directory contains publicly released security advisories, notices, vulnerability disclosures, CVE statements, and related security publications.

## Advisory Types

### Security Advisories

Issues confirmed by the TechSmith Security Team that may affect products, services, or customers.

### CVE Statements

Public statements regarding CVE assignments, dispute positions, clarifications, corrections, or related matters.

### Security Notices

Informational publications that may be security relevant but do not necessarily represent a vulnerability.
